# Game Design Prompt

Create a complete 2D platformer game in **Godot 4.x** that combines the nostalgic exploration of **Dangerous Dave**, the unpredictable troll mechanics of **Level Devil**, the precision platforming of **Celeste**, and the fast-paced satisfaction of modern mobile platformers.

The game should be original and should not copy any existing game levels, artwork, characters, or assets.

---

# Core Concept

A deceptively simple platformer where the player believes each level is straightforward, but the environment constantly surprises them.

The goal is to create a game that is:

* Easy to understand
* Difficult to master
* Funny
* Frustrating in a fair way
* Highly replayable
* Addictive
* Speedrun friendly

Every death should teach the player something new.

# Level Design Philosophy

Each level introduces ONE new mechanic.

Then combine mechanics gradually.

Difficulty should increase naturally.

No impossible jumps.

Every trap should be avoidable after learning.

Players should laugh after dying.

# Goal

Build a polished indie-quality platformer with hundreds of handcrafted levels, satisfying movement, hilarious trap mechanics, rewarding exploration, competitive leaderboards, cloud saves, achievements, and seasonal content. The game should encourage players to say "Just one more level," while rewarding skill, experimentation, and mastery through fair but surprising challenges.
